# Privacy Policy — Minimal LinkedIn

_Last updated: 2026-07-13_

**Minimal LinkedIn does not collect, store, transmit, or share any data.**

The extension runs entirely on your own device, inside your browser. It has no
servers and makes no network requests of its own.

## What the extension does

Minimal LinkedIn injects a small content script on `linkedin.com` pages whose
only job is to hide the home-page feed (and show a short "feed hidden" note).
That's it.

## Data collection

- **No personal data** is collected or processed.
- **No browsing history, messages, contacts, or profile information** is read,
  stored, or transmitted.
- **No analytics, telemetry, or tracking** of any kind.
- **No cookies** are set by the extension.
- **No data is sent to the developer or any third party.**

## Permissions

The extension requests a single host permission, `*://*.linkedin.com/*`. This is
required solely so its content script can run on LinkedIn pages to hide the
feed. It is not used to read or exfiltrate any information.

## Remote code

The extension contains no remote code. All logic ships inside the extension
package and runs locally.

## Changes

If this policy ever changes, the updated version will be published in this
repository.
